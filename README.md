# Dict Parser

### Tested Platforms

- Ubuntu 14.04, with g++ 4.8.4

### Dependencies

- Google Test

### How to build

    mkdir build
    cd build
    cmake ..
    make

### How to run tests

    cd build/test
    ./built_in_parser_test
    ./dict_parser_test

or

    cd build/test
    ctest

### How to run demo

    cd build/demo
    ./dict_parser_demo
